# Rust学习资源汇总

## 官方资源

### 基础学习
- [The Rust Programming Language](https://doc.rust-lang.org/book/) - Rust官方教程

### 进阶学习

## 学习策略

### 每日学习
1. **晨读** (30分钟): 阅读官方文档或教程
2. **编码实践** (2-3小时): 完成项目或练习
3. **复习总结** (30分钟): 记录学习笔记

### 每周复习
1. 回顾本周学习内容
2. 完成一个小项目
3. 参与社区讨论
4. 阅读相关技术文章

### 每月评估
1. 完成一个综合项目
2. 评估学习进度
3. 调整学习计划
4. 寻找实习或项目机会
